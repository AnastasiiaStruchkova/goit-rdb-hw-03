# Домашнє завдання 3: Завантаження даних та основи SQL. DQL команди

## Опис проєкту
Цей репозиторій містить виконання домашнього завдання до Теми 3 з курсу реляційних баз даних. Головна мета проєкту — відпрацювання навичок написання DQL (Data Query Language) запитів для отримання та аналізу даних із завантаженого датасету електронної комерції.

## Виконані завдання
У межах цього домашнього завдання були написані SQL-запити для вирішення наступних задач:
1. Вибірка всіх даних із таблиці `products` та специфічних колонок із таблиці `shippers`.
2. Використання агрегатних функцій `MIN`, `MAX` та `AVG` (із заокругленням `ROUND`) для розрахунку статистики цін товарів.
3. Використання оператора `DISTINCT` для пошуку унікальних комбінацій категорій та цін, сортування `ORDER BY` та лімітування результатів через `LIMIT`.
4. Пошук товарів у заданому ціновому діапазоні за допомогою оператора `BETWEEN`.
5. Групування даних за допомогою `GROUP BY` для підрахунку кількості товарів та середньої ціни для кожного постачальника.
